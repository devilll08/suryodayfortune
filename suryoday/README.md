# Suryoday Fortune Website

This project has been refactored from a single-file HTML page into a professional static website structure.

## Directory Structure

- index.html
- suryoday.html
- assets/
- assets/css/style.css
- assets/js/main.js
- assets/images/

## Run Locally

Option 1:
- Open index.html directly in a browser.

Option 2 (recommended):
- Run a local static server from this folder.
- Example: python -m http.server 5500
- Visit http://localhost:5500
